Sobre o NodeBR

O NodeBR.com é o repositório de artigos da comunidade brasileira de node.js, seu propósito é ensinar como desenvolver soluções em node assim como ensinar os conceitos fundamentais necessários para escrever um código eficiente.

Este portal é gerado automáticamente pela engine jekyll, utiliza o twitter bootstrap como framework de front-end e é hospedado gratúitamente pelo github pages.

Todo o conteúdo deste portal é armazenado em um repositório git onde qualquer pessoa pode dar um fork, escrever um artigo e enviar uma requisição de pull. Se o artigo passar nos padrões de qualidade ele será publicado e ajudará a crescente comunidade brasileira de node.
